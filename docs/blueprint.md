# **App Name**: Al Batul Matrimony

## Core Features:

- Intelligent Matchmaker Tool: Uses AI as a tool to suggest compatible profiles by analyzing sect, education, lifestyle, and city preferences to improve matchmaking quality.
- Detailed Matrimonial Profiles: Complete profile management including sect, age, marital status, and education with Firestore-based secure photo uploads.
- Interest Exchange Protocol: Formal interest request system where users can express and accept interest before initiating a conversation.
- Encrypted Private Messaging: A secure chat interface allowing accepted matches to communicate privately in real-time.
- Manual UPI Payment Verification: User portal for uploading payment screenshots for premium membership, handled through an admin verification queue.
- Profile Verification and Trust Center: Dedicated verification process for user identity and status to ensure a secure and trusted community for all members.
- Admin Moderation Suite: A dashboard for admins to approve profiles, verify payments, and manage block/report logs.

## Style Guidelines:

- Primary Color: Deep Sage (#478572) representing growth and tradition. A light scheme is used for clarity and elegance.
- Background Color: Subtle Mist (#F2F7F5), providing a clean and open environment that evokes purity.
- Accent Color: Soft Leaf (#82C98E) for interest requests and positive feedback loops.
- Font Pairing: 'Alegreya' (Serif) for headlines to provide a literary and respectful feel, paired with 'PT Sans' (Sans-serif) for modern and legible body content.
- Clean, thin-line geometric icons that feel professional and modern, avoiding overly literal religious symbols.
- A refined, centered masonry grid for profile discovery, emphasizing photography and high-level vitals.
- Gentle cross-fade transitions between search filters and profile details to maintain a serene user experience.
